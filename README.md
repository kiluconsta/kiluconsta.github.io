# The Vault

A personal media vault hosted via GitHub Pages.

## Setup

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site will be live at `https://yourusername.github.io/your-repo-name`

## Adding content

Open `index.html` and find the `SECTION REGISTRY` comment near the top of `<body>`. Instructions are inline — copy one of the two templates (video or image) and fill in your URLs.

## Structure

Everything is in a single `index.html` file. No build step, no dependencies to install — just HTML, CSS, and vanilla JS. The only network requests are Google Fonts (loaded in `<head>`) and your media URLs.
